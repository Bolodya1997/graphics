Игра "Жизнь":
    Есть множество клеток, расположенных в сетке из правильных 
шестиугольников, каждая из которых в заданный момент времени может 
находиться в одном из двух состояний: живая или мертвая. Каждый такт 
игры состояние клеток пересчитывается на основании состояния соседей
по некоторому заранее заданному алгоритму.

Элементы приложения (сверху вниз):
	1.	панель меню (menu bar);
	2.	панель инструментов (toolbar);
	3. 	игровая панель - поле, с размеченной на нём сеткой из правильных
		шестиугольников;
	4.	панель состояния (status bar);

Изменение параметров игры:
    Изменить параметры игры можно кнопкой (Grid settings) на панели
инструментов или в меню (Edit).
    Параметры и их значение:
	1.	Field properties - настройки игровой панели:
		1.1.	Rows				количество ячеек в строке
		1.2.	Columns			количество ячеек в высоту
		1.3.	Cell size			длина ребра ячейки (по внутренней стороне)
		1.4.	Grid width		ширина ребра ячейки
	2.	Mode - шаблоны поведения (см. Изменение состояния ячеек):
		2.1.	Replace
		2.2.	XOR
	3.	Impact - величины, задающие влияние соседей на клетку:
		3.1.	FST_IMPACT		соседей первого порядка
		3.2.	SND_IMPACT		соседей второго порядка
	4.	Environment - параметры алгоритма пересчёта состояния клетки:
		4.1.	LIVE_BEGIN
		4.2.	LIVE_END
		4.3.	BIRTH_BEGIN
		4.4.	BIRTH_END

Изменение состояния ячеек:
    При нажатии на ячейки решетки, они могут изменять своё состояние по
одному из двух шаблонов поведения:
	1.	replace:		живая -> живая			мертвая -> живая;
	2.	XOR:			живая -> мертвая		мертвая -> живая;
    По умолчанию задаётся режим "replace", изменить состояние можно 
кнопкой (XOR | Replace) на панели инструментов, в меню (Edit) или в 
настройках параметров игры.

Игра:
    Такты игры можно реализовывать пошагово, нажатием на кнопку (Step) 
на панели инструментов или в меню (Action), или по таймеру, управляя им 
нажатием на кнопку (Play | Stop) на панели инструментов или в меню 
(Action).

Дополнительные возможности к просмотру состояния ячеек:
	1.	Отображение влияния соседей на ячейку в виде числа на ячейке - 
		управляется кнопкой (Impact) на панели инструментов или в меню
		(View).
	2.	Отображение цветом не только текущего состояния ячейки, но и 
		состояния на следующий такт - управляется кнопкой (Colors) на 
		панели инструментов или в меню (View).

Загрузка, сохранение модели с помощью файлов:
    Новое игровое поле может быть загружено из файла с помощью кнопки 
(Open) на панели инструментов или в меню (File).
    Текущее игровое поле можно сохранить в файл с помощью кнопки (Save) 
на панели инструментов или в меню (File).
