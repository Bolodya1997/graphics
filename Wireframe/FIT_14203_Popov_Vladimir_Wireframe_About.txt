Попов Владимир ФИТ 14203

Приложение для работы с 3D фигурами вращения.

Дополнительные параметры:
	'4'		:	движение по оси 0x вперед;
	'1'		:	движение по оси 0x назад;
	'5'		:	движение по оси 0y вперед;
	'2'		:	движение по оси 0y назад;
	'6'		:	движение по оси 0z вперед;
	'3'		:	движение по оси 0z назад;
